============================================================
   CANDLE QUEST ACADEMY  -  How to start the game
============================================================

This is a game. No installing. Nothing to buy. It uses pretend
"Koins" only -- never real money.


------------------------------------------------------------
 THE EASIEST WAY (any device)
------------------------------------------------------------
   Open this link in any web browser:

       https://crystalball-seven.vercel.app/

   Works on Mac, Windows, phones and tablets. On a Mac you can
   drag the address-bar icon to your Desktop or Dock to keep it.


------------------------------------------------------------
 PUT A CLICKABLE ICON ON THE DESKTOP (with the cool icon)
------------------------------------------------------------
 WINDOWS:
   Double-click   Make Desktop Icon (Windows).bat
   -> a "Candle Quest" icon appears on the Desktop.

 MAC:
   Double-click   Make Desktop Icon (Mac).command
   (First time, right-click the file -> Open -> Open to allow it.)
   -> a "Candle Quest" icon appears on the Desktop.

 Either icon opens the live game in your browser.
